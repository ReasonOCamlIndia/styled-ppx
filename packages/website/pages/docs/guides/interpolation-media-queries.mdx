## Interpolation in Media queries

```rescript
module Media = {
  let mobileDown = "(max-width: 767px)"
}

let container = %cx(`
  display: grid;
  grid-column-gap: 48px;
  grid-template-columns: repeat(2, 1fr);
  width: 100%;

  @media $(Media.mobileDown) {
    grid-template-columns: 100%;
  }
`)
```
