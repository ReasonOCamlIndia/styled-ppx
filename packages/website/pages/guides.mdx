# Guides

## Nested selectors

```rescript
let link = %cx(`
  color: $(Color.Text.body);

  &:hover {
    color: $(Color.Text.accent);
  }
`)
```

## Interpolation in selectors

```rescript
let highlighted = %cx(`
  padding: 4px;
  outline: 1px solid red;
`)

let link = %cx(`
  color: $(Color.Text.body);

  & .$(highlighted) {
    padding: 0px;
  }
`)
```

## Interpolation in Media queries

```rescript
module Media = {
  let mobileDown = "(max-width: 767px)"
}

let container = %cx(`
  display: grid;
  grid-column-gap: 48px;
  grid-template-columns: repeat(2, 1fr);
  width: 100%;

  @media screen and $(Media.mobileDown) {
    grid-template-columns: 100%;
  }
`)
```
<small>Note that all interpolation in preludes (*selectors/media queries/etc*) are strings
and only the structure is being parsed and validated.</small>
